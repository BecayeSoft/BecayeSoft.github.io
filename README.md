# Becaye's Portfolio & Blog
My Portfolio and Blog built with Angular.

**Warning ⚠️:** CSS stops working after deploying for the first time.

## Deploying on Firebase
```
ng build --prod
firebase deploy
```

## Deploying on github
```
ng build --prod
```
Put the content of the dist/becayesoft-portfolio-blog/ inside becayesoft.github.io/docs/
